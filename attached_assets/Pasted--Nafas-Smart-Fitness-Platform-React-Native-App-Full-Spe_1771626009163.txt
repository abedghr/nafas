# نَفَس (Nafas) — Smart Fitness Platform
## React Native App — Full Specification for Replit

---

## Project Overview

**Nafas** is an AI-powered sports and fitness platform that unifies community, training, nutrition, coaching, tournaments, and real-world gym/restaurant discovery into one seamless mobile experience.

- **App Name:** نَفَس (Nafas)
- **Platform:** React Native (iOS + Android)
- **Languages Supported:** Arabic 🇸🇦 and English 🇬🇧 (full i18n support)
- **Data:** Use mock data for all features
- **Design Style:** Modern, dark/light theme toggle, sports-focused UI

---

React Native (Expo preferred)
React Navigation (Bottom Tab + Stack + Drawer)
React Native Maps
i18n / react-i18next (Arabic + English)

## 4. AI Coach (Main Tab)

This tab is the user's **personal training hub**.

### 4.1 Dashboard
- Today's workout plan (AI-generated or manual)
- Quick stats: weekly workouts done, total volume, streak
- AI tip of the day (e.g., "Your energy was lower yesterday — consider a deload today")

---

### 4.2 Start Workout

**Step 1 — Select Workout Type:**
```
Push Day | Pull Day | Leg Day | Upper Body | Full Body | Cardio | Custom
```

**Step 2 — Workout Logger:**
- Add exercises (search from mock exercise library)
- For each exercise: sets, reps, weight (kg/lbs)
- Pre-workout toggle: "Did you take pre-workout? 💊"
- Rest timer between sets
- Notes field

**Step 3 — Finish & Save:**
- Summary: total volume, duration, exercises done
- Option: **Share to Social / Community**
- Workout saved to history

---

### 4.3 Workout History & Progress

- Calendar view of past workouts
- Weekly / Monthly volume charts
- Per-exercise progress (e.g., Bench Press progression over time)
- Compare with last week

---

### 4.4 InBody / Body Composition Tracker

- Upload InBody test results (mock image or manual entry)
- Fields: Weight, Muscle Mass, Body Fat %, BMI, Water %
- Visual chart: progress over months
- Insights:
  - Best performance period
  - Monthly comparison
  - Annual average

---

### 4.5 AI Smart Training Plan

**User input:** "I want to learn Handstand in 4 months"

**AI asks:**
- Current level? (Beginner / Intermediate / Advanced)
- How many training days per week?

**AI generates monthly plan:**
```
Month 1: Shoulder strengthening & wrist conditioning
Month 2: Wall-assisted holds & balance drills
Month 3: Wall control & kick-up practice
Month 4: Free-standing hold attempts
```

**AI Warnings:**
- "Most people who rush this skill get injured — reduce speed"
- Compare with successful user journeys

**AI Suggestions:**
- If plan seems unrealistic, AI flags it and recommends adjustment

---

### 4.6 AI Features (Smart Insights)

| Feature | Description |
|---|---|
| AI Personal Coach | Daily energy & performance tips |
| AI Mistake Detection | Warns about high injury-risk patterns |
| AI Nutrition Optimizer | Suggests macro adjustments based on workout intensity |
| AI Community Matching | "This user has the same goal and level as you" |
| AI Trend Detection | "Handstand Challenge is up 40% this week" |

## 5. Nutrition (Main Tab)

### 5.1 Daily Nutrition Dashboard

- Daily macro targets (calculated from user profile + goal)
- Progress bars: Protein / Carbs / Fats / Calories
- Starts at 0 every morning
- Meals log: Breakfast, Lunch, Dinner, Snacks

**Macro Calculation (mock logic):**
```
Protein: weight (kg) × 2g  (for muscle gain)
Carbs: remaining calories / 4
Fats: 20-25% of total calories
```

---

### 5.2 Log a Meal

- Search food from mock food database
- Common foods pre-loaded: eggs, chicken breast, rice, oats, etc.
- Each food shows: protein, carbs, fats, calories
- Add portion size → auto-calculates macros
- Option to add meal from **Healthy Restaurants** menu

---

### 5.3 21-Day Commitment Challenge

- User opts into a 21-day nutrition plan
- Each day resets at midnight
- End of 21 days: full compliance report
  - Days fully on track ✅
  - Days where goals were missed ❌
  - Streak count 🔥

---

### 5.4 End-of-Day Summary

- Pie chart: macros consumed
- Compliance percentage
- AI suggestion: "You hit 95% today! Great job 💪"

### 6.6 Ready to Train / Find a Partner

**How it works:**
1. User taps "I'm Ready to Train 🟢"
2. Selects activity: Running / Gym / Calisthenics / Football / etc.
3. App shows nearby users who are also "Ready"
4. Both users tap **Accept**
5. Location / meeting point confirmed (mock)
6. Shared workout session is logged for both users

**States:**
- 🟢 Ready
- 🔴 Not Available
- 🕐 Scheduled (set future time)

## Important Notes for Replit

1. **Use mock data only** — no real backend required. Load all data from local JSON files.
2. **React Navigation** — use Bottom Tab Navigator for main 5 tabs, Stack Navigator for inner screens, Drawer Navigator for community sidebar.
3. **RTL Support** — when language is Arabic, flip layout direction using `I18nManager.forceRTL(true)`.
4. **Map** — use `react-native-maps` with mock coordinates for gyms, restaurants, and tournament venues.
5. **Charts** — use `react-native-chart-kit` or `victory-native` for progress charts.
6. **Image placeholders** — use `https://picsum.photos` or `https://source.unsplash.com` for all mock images.
7. **Start with these screens first:** Onboarding → Communities → AI Coach (workout logger) → Nutrition Dashboard → Profile.
8. **AI responses** — simulate AI by returning pre-written mock responses based on user input keywords.
9. **Storage** — use AsyncStorage to persist user session, workout logs, and nutrition logs across sessions.
10. **Language toggle** — add toggle in Profile settings to switch between Arabic and English instantly.
